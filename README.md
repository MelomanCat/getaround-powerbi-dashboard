# Getaround Rental Operations Dashboard

## Business Problem

Getaround experiences rental conflicts when vehicles are returned late.
The objective is to evaluate the impact of introducing a minimum buffer between rentals.

## Key Findings

- 37% of rentals experienced a significant delay (>30 min).
- Early returns represent the largest category of rentals.
- A 60–90 minute buffer provides the best trade-off between prevented conflicts and operational impact.
